# Million-Square

คำเเนะนำในการ installเเละรัน:
install ไฟล์งานลงเครื่องของตนเอง เปิดโปรเเกรมเละเข้าไฟล์จากนั้นกดปุ่มรัน เเละเข้าหน้าเว็บ http://localhost:8090/home จากนั้นทำการสั่งได้
